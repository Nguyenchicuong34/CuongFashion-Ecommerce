![demoshop](https://github.com/user-attachments/assets/840fc07f-d55c-4120-9ed3-e7f84a0d8e29)#  Cuong Fashion - Hệ thống Thương mại điện tử

Dự án website bán hàng thời trang full-stack được xây dựng bởi **Nguyễn Chí Cường**.

##  Công nghệ sử dụng
- **Backend:** Python (Django Framework)
- **Frontend:** HTML5, CSS3, Bootstrap 5
- **Database:** SQLite / SQL Server
- **Tính năng:** - Mua hàng, Giỏ hàng (Session-based)
  - Thanh toán QR Code (VietQR API)
  - Gợi ý sản phẩm thông minh
